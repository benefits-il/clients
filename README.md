# benefits-il / clients

תיקיית-מסירה לתוצרי-לקוח של קורס Benefits — עמודי-עבודה, דפי-תרגול וחומרים מוכנים-להעתקה שהמשתתפים פותחים בלינק חי.

## מבנה
- תיקייה לכל לקוח: `<שם-לקוח>/`
- כל עמוד = קובץ HTML עצמאי אחד (כל ה-CSS וה-JS בפנים, גופנים ואייקונים מ-CDN).
- נפרס דרך GitHub Pages (main/root) ← `benefits-il.dev/clients/<שם-לקוח>/`.
